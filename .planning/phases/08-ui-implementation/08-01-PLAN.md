## Tasks

### Task 1: Create ThinkingBubble composable

**File:** `app/src/main/java/com/gemma/gpuchat/MainActivity.kt`

**Action:**
Add `ThinkingBubble` composable after the existing `ChatBubble` composable (after line 1467). This composable:

1. Takes `thinkingText: String` as state parameter (for streaming updates)
2. Uses `surfaceVariant` background color with rounded corners (topStart=12.dp, topEnd=12.dp, bottomStart=4.dp, bottomEnd=4.dp)
3. Smaller text style: `MaterialTheme.typography.bodySmall`
4. "thinking..." label prefix in a muted color (`MaterialTheme.colorScheme.onSurfaceVariant.copy(alpha=0.7f)`)
5. `maxLines = 4` with `TextOverflow.Ellipsis` for clamped state
6. Expand/collapse state managed via `var isExpanded by remember { mutableStateOf(false) }`
7. "show/hide" `TextButton` that toggles `isExpanded`
8. `AnimatedVisibility` wraps the content for smooth transitions:
   - `EnterTransition.FadeIn` + `ExitTransition.FadeOut`
   - Duration 300ms for enter, 200ms for exit

```kotlin
@Composable
fun ThinkingBubble(thinkingText: String) {
    if (thinkingText.isEmpty()) return

    var isExpanded by remember { mutableStateOf(false) }

    Column(
        modifier = Modifier
            .fillMaxWidth()
            .padding(bottom = 4.dp),
        horizontalAlignment = Alignment.Start
    ) {
        Box(
            modifier = Modifier
                .widthIn(max = 260.dp)
                .background(
                    color = MaterialTheme.colorScheme.surfaceVariant,
                    shape = RoundedCornerShape(
                        topStart = 12.dp,
                        topEnd = 12.dp,
                        bottomStart = 4.dp,
                        bottomEnd = 4.dp
                    )
                )
                .padding(8.dp)
        ) {
            Column {
                Text(
                    text = "thinking...",
                    style = MaterialTheme.typography.labelSmall,
                    color = MaterialTheme.colorScheme.onSurfaceVariant.copy(alpha = 0.7f),
                    modifier = Modifier.padding(bottom = 2.dp)
                )
                Text(
                    text = thinkingText,
                    style = MaterialTheme.typography.bodySmall,
                    color = MaterialTheme.colorScheme.onSurfaceVariant,
                    maxLines = if (isExpanded) Int.MAX_VALUE else 4,
                    overflow = TextOverflow.Ellipsis
                )
                if (thinkingText.lines().size > 4) {
                    TextButton(
                        onClick = { isExpanded = !isExpanded },
                        modifier = Modifier.padding(top = 2.dp)
                    ) {
                        Text(
                            text = if (isExpanded) "hide" else "show more",
                            style = MaterialTheme.typography.labelSmall
                        )
                    }
                }
            }
        }
    }
}
```

**Verify:** `grep -c "fun ThinkingBubble" app/src/main/java/com/gemma/gpuchat/MainActivity.kt` returns `1`

**Done:** `ThinkingBubble` composable exists with muted styling (surfaceVariant, bodySmall text, "thinking..." label), 4-line clamp, expand/collapse toggle, and AnimatedVisibility transitions.

### Task 2: Wire onThinking callback in all sendMessage() calls

**File:** `app/src/main/java/com/gemma/gpuchat/MainActivity.kt`

**Action:**
For each of the 6 `sendMessage()` call sites (lines 402, 621, 658, 1039, 1190, 1304), add the `onThinking` callback parameter. The callback accumulates thinking text into the last bot message's `thinkingText` field.

**Pattern for each sendMessage() call:**

Before:
```kotlin
LlmChatModelHelper.sendMessage(
    message = ...,
    onToken = { token -> ... },
    onDone = { ... },
    onError = { ... }
)
```

After (add `onThinking` parameter):
```kotlin
LlmChatModelHelper.sendMessage(
    message = ...,
    onToken = { token -> ... },
    onDone = { ... },
    onError = { ... },
    onThinking = { thinking ->
        mainHandler.post {
            val lastBot = messages.indexOfLast { !it.isUser }
            if (lastBot >= 0) {
                messages = messages.mapIndexed { idx, msg ->
                    if (idx == lastBot) msg.copy(thinkingText = thinking) else msg
                }
            }
        }
    }
)
```

**Important:** Use `mainHandler.post {}` for UI updates — LiteRT-LM callbacks run on background threads per Phase 7 threading contract.

**Verify:** `grep -c "onThinking = {" app/src/main/java/com/gemma/gpuchat/MainActivity.kt` returns `6`

**Done:** All 6 `sendMessage()` calls have `onThinking` callback that accumulates thinking text into the last bot message's `thinkingText` field using `mainHandler.post {}`.
