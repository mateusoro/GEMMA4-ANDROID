## Milestone Summary

### Key Decisions
- **jvmToolchain(17) + multiDex:** Nubia Flip 5G ART required Java 17 toolchain, MainActivity must be in primary DEX
- **Prepend system prompt:** Gemma-4-E2B-IT ignores `ConversationConfig.systemInstruction` → workaround in `sendMessage()`
- **`LlmChatModelHelper.reload()` pattern:** Settings changes must call reload() (not updateParams()) to recreate engine
- **TestHarnessActivity in app/src/main/:** Bypasses AGP JDK 24/21 classpath mismatch in test runner

### Issues Resolved
- `ClassNotFoundException: MainActivity` on Nubia Flip 5G → `jvmToolchain(17)` + multiDex fix
- "Nova conversa" responded "fixed laptop" → `resetConversation()` not called on new chat
- Settings not applied on startup → duplicate `LaunchedEffect` removed, `DisposableEffect` release removed
- System prompt not in Portuguese → added "Responda sempre em português. O nome do usuário é Mateus."

### Issues Deferred
- None — all shipped in v1.0
